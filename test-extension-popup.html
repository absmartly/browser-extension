Manual Extension Test Required

To properly test the avatar display in the extension:

  1. Open Chrome
  2. Go to chrome://extensions/
  3. Enable "Developer mode"
  4. Click "Load unpacked"
  5. Select the folder:
    /Users/joalves/git_tree/absmartly-ext/absmartly-browser-extension/build/chrome-mv3-prod
  6. Click the extension icon in toolbar
  7. Click "Settings"
  8. Wait 5 seconds for auth check
  9. Check if avatar appears

What to look for:

  • A circular user avatar image next to the user name
  • The avatar should be 40x40 pixels
  • No broken image icon

Debug in Console:

Right-click extension popup > Inspect > Console

Look for:

  • "Avatar fetched successfully" (success)
  • "Failed to fetch avatar" (failure)

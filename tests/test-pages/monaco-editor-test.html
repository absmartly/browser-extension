Test Scenarios

Use these scenarios to verify the Monaco editor features:

  1. Syntax Highlighting: HTML tags should be color-coded
  2. Autocomplete: Type "<" to see tag suggestions
  3. Format Document: Click the Format button to auto-indent HTML
  4. Wrap Selection: Select text and click Wrap to surround with tags
  5. Preview: Click Preview to see live changes before applying
  6. Keyboard Shortcuts: ESC to cancel, Shift+Alt+F to format

✅ Test Checklist

Verify these Monaco features are working:

  1. Syntax Highlighting: HTML tags appear in different colors (blue/purple for tags, orange for attributes, green for strings)
  2. Autocomplete: Type < and see tag suggestions. Type space after a tag name for attribute suggestions
  3. Format Document: Click the Format button to auto-indent messy HTML
  4. Dark Theme: Editor should have a dark background matching VS Code
  5. Line Numbers: Line numbers should appear on the left
  6. Wrap Selection: Select text and click Wrap to surround with <div> tags
  7. Preview: Click Preview to temporarily see changes
  8. Save/Cancel: Save applies changes, Cancel discards them
  9. ESC Key: Press ESC to quickly close the editor
